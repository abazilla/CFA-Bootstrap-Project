JUST 3 QUICK STEPS

There is zero installation required. Just extract and zip the archive and you are already up and running.

Install

In a nutshell, installing Grav is a matter of unzipping the file to your web server. Yay!

Build

Content is just simple markdown files in folders, with no complicated database to deal with.

Maintain

A powerful API and extensive hooks allow for plugins to extend Grav, such as the Admin Plugin.

FROM SIMPLE TO SOPHISTICATED

Grav has the flexibility to power all sorts of websites. Flexible Twig templating allows easy realization of your designs

CRAZY FAST

EASY TO WORK WITH

FREE AND OPEN
